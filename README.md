# TanjaMol Storefront

Arabic COD storefront prototype for TanjaMol.

## Local testing

```bash
npm install
npm run dev
```

Open:

- Storefront: http://127.0.0.1:5173/#/
- Product page: http://127.0.0.1:5173/#/product/smart-watch

## Production check

```bash
npm run build
npm run preview
```

## GitHub Pages

```bash
npm run deploy:gh-pages
```

This builds the site and publishes the `dist` folder to the `gh-pages` branch.
